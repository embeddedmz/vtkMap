vtkMap is a project wich adds VTK support for Geovisualization in 2D and 3D.

It's a slippy map that can be used in C++/Qt programs.

Requirements
------------
* C++11
* VTK
* OpenGL 3.2

Additional dependencies
-----------------------
Some extensions and applications might require:
* GDAL
* Qt5
